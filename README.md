# SALES_DASHBOARD_2026_POWER-BI

## 📌 Project Overview

This project is an interactive Sales Analytics Dashboard developed using Power BI to analyze sales performance, customer behavior, and product insights across multiple countries and categories.

The dashboard helps businesses monitor key KPIs, identify top-performing products, and track sales trends over time through interactive visualizations and filters.

---

# 🎯 Project Objective

The main objective of this project is to:

* Track overall sales performance
* Analyze customer and product trends
* Monitor revenue growth
* Compare sales across countries and categories
* Create an interactive business intelligence dashboard for better decision-making

---

# 📊 Dashboard Features

✅ KPI Cards for:

* Total Orders
* Total Revenue
* Total Customers

✅ Sales Trend Analysis

* Quarterly and yearly sales performance visualization

✅ Product Category Analysis

* Sales distribution across categories

✅ Top Products Analysis

* Best-selling product comparison

✅ Country-wise Sales Analysis

* Interactive map visualization

✅ Dynamic Filters & Slicers

* Country filter
* Product category filter
* Date range filter

---

# 📈 Key Insights

* Laptop category generated the highest sales contribution.
* Dell XPS 15 emerged as the top-selling product.
* Sales showed strong growth trends over time.
* Country-based analysis highlighted strong sales performance in India, China, and the United States.
